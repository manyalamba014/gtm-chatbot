# GTM Strategy Assistant

An AI-powered chatbot that helps sales, marketing, and revenue teams build smarter go-to-market strategy — from ICP development to cold outreach to positioning.

Built with the Anthropic Claude API as a portfolio project for GTM and AI Engineer roles.

---

## What it does

- **ICP development** — define your ideal customer profile with firmographic and behavioral criteria
- **Cold outreach** — generate personalized email sequences, LinkedIn messages, and call scripts
- **Positioning & messaging** — build messaging frameworks, one-pagers, and battlecards
- **Discovery call prep** — surface the right questions for any buyer persona
- **Sales strategy** — apply frameworks like MEDDICC, SPIN Selling, and StoryBrand

## Modes

Switch between four specialized modes depending on what you're working on:

| Mode | Best for |
|------|----------|
| General | Broad GTM strategy, frameworks, market sizing |
| BDR / Sales | Cold outreach, objection handling, pipeline building |
| Product marketing | Positioning, messaging, competitive intel, launch planning |
| RevOps | Pipeline metrics, lead scoring, territory design, forecasting |

---

## Getting started

1. Clone or download this repo
2. Open `index.html` in any browser — no server or install required
3. Get a free API key at [console.anthropic.com](https://console.anthropic.com)
4. Paste your key into the yellow banner when the app loads
5. Start chatting

Your API key is stored only in your browser's local storage and is never sent anywhere except directly to Anthropic.
